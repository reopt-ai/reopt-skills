# opt-harness-install

Install or upgrade `@reopt-ai/opt-harness`. Generates the harness manifest, wraps the app in `HarnessProvider`, and creates the first Workspace page from a recipe.

## Use When

- a consumer project needs an app shell, workspace layouts, or density/navigation policy
- bootstrapping Dashboard/List/Detail/Editor/Landing workspace pages
- upgrading opt-harness and want the impact scan + theme verification
