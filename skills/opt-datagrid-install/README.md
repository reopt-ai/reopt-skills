# opt-datagrid-install

Install, upgrade, or migrate to `@reopt-ai/opt-datagrid`. Auto-detects glide-data-grid, ag-grid, react-data-grid, and MUI DataGrid usage and plans a conversion.

## Use When

- a new consumer project needs a DataGrid with type-safe columns and editors
- upgrading opt-datagrid and want to run the breaking-change scan + type-check loop
- migrating away from another grid library to `@reopt-ai/opt-datagrid`
