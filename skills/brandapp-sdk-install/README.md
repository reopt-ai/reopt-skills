# brandapp-sdk-install

Install `@reopt-ai/brandapp-sdk` in a consumer project: .npmrc, env, SDK instance, Better Auth, EAV schema, webhooks.

## Use When

- a Next.js project is adopting `@reopt-ai/brandapp-sdk` for the first time
- OAuth, EAV, Files, or AI via the unified SDK needs scaffolding
- CI or local dev needs an offline dev server (`REOPT_DEV_MODE`)
