# opt-ui-install

Install, upgrade, or add a Surface to `@reopt-ai/opt-ui` in a consumer project. Wires Tailwind CSS v4, `OptThemeProvider`, the theme boot script, and the Surface CLI.

## Use When

- a new consumer project needs Tailwind v4 + opt-ui theme set up
- upgrading opt-ui and need the 26-check doctor plus breaking-change fixes
- adding a page template via `npx @reopt-ai/opt-ui-cli add <slug>`
