# opt-editor-install

Install or upgrade `@reopt-ai/opt-editor` in a consumer project. Generates the block catalog, Editor component, CSS import, and optional AI streaming wiring.

## Use When

- setting up `@reopt-ai/opt-editor` for the first time (catalog + Editor component)
- upgrading opt-editor and running the 18-check doctor + breaking-change fixes
- adding AI streaming (`--with-ai`) to an existing editor
