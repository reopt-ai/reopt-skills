# opt-chat-install

Install or upgrade `@reopt-ai/opt-chat` in a consumer project. Wires `.npmrc`, Tailwind content scan, AI SDK endpoint, and a starter Conversation component.

## Use When

- a project is adding chat UI powered by `@reopt-ai/opt-chat`
- bumping opt-chat across breaking changes and need the impact scan
- you need a minimal Vercel AI SDK endpoint + Conversation scaffold
